Звездное небо

запуск GUI.py
в левом поле вводится время, в правом угол наклона головы
можно ввести данные вручную или изменять с помощью стрелок
с помощью колесика мыши можно менять масштаб

В файлах stars, geometry, drawer находится логика соответствующих объектов:
stars - классы звезд, созвездий и измерений их координат
geometry - класс для расчета координат для изображения
drawer - класс для создания изображения неба
