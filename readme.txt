Звездное небо

запуск pygame_GUI.py
с помощью кнопок-стрелок можно поворачивать поле зрения в соответствующие стороны


В файлах stars, geometry, drawer находится логика соответствующих объектов:
stars - классы звезд и измерений их координат
geometry - класс для расчета координат для изображения
drawer - класс для получения визульных характеристик звезд (цвет и радиус)
